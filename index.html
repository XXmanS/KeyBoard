Что такое клавиатура

ᅟᅟᅟᅟᅟᅟКлавиатура — это устройство ввода, позволяющее пользователю передавать команды и вводить информацию в компьютер с помощью набора клавиш (кнопок), расположенных в определённом порядке.

    Клавиатура нужна для:

  • ввода текста (букв, цифр, знаков препинания);
  • выполнения команд и операций (через функциональные и специальные клавиши);
  • управления курсором и навигации по интерфейсу;
  • использования сочетаний клавиш для ускорения работы;
  • управления мультимедийными функциями (в современных моделях).

Эволюция клавиатуры

Клавиатура была не сразу для компьютера. Впервые она появилась в 1714 году. Она была предназначена для устройства, похожего по описанию на пишущую машинку, она не отсоединялась и не было отдельным комплектом. Сама пишушая машинка была создана английским инженером Генри Милль.

Первая клавиатура для компьютера была создана в 1964 году. Она была сделана "IBM" для своего терминала "IBM 2741"

После первой быыла создана вторая (Model F), специально для компьютера "IBM PC", Она была оснащена 83 клавишами и механизмом емкостных переключателей

"IBM" представляет улучшенную клавиатуру "Model M" в 1986 году, она была оснащена 101 клавишей

Сейчас выходят более мощные компьютеры и клавиатуры и все знают их внешний вид

Горячие клавиши

    Системные сочетания (работа с ОС)
  • Win — открыть/закрыть меню «Пуск»;
  • Win + D — показать/скрыть рабочий стол;
  • Win + E — открыть «Проводник»;
  • Win + L — заблокировать компьютер;
  • Win + M — свернуть все окна;
  • Win + Shift + M — развернуть свёрнутые окна;
  • Win + R — открыть окно «Выполнить»;
  • Win + I — открыть «Параметры» системы;
  • Win + A — открыть центр уведомлений / панель быстрых настроек;
  • Win + S — поиск по системе;
  • Win + Tab — обзор задач (все окна и рабочие столы);
  • Win + V — история буфера обмена;
  • Win + K — подключение беспроводных устройств;
  • Win + X — меню опытного пользователя (быстрый доступ к управлению дисками и т. д.);
  • Win + Pause — открыть окно «Система» (сведения о ПК);
  • Win + P — переключение режимов отображения (второй монитор, проектор);
  • Win + . или Win + ; — открыть панель эмодзи;
  • Ctrl + Shift + Esc — открыть диспетчер задач;
  • Ctrl + Alt + Del — экран безопасности (блокировка, смена пользователя, диспетчер задач);
  • Alt + Tab — переключение между окнами;
  • Alt + Esc — переключение окон в порядке открытия (без предварительного просмотра);
  • Alt + F4 — закрыть активное окно (на рабочем столе — вызов меню завершения работы);
  • Alt + Space — открыть системное меню окна (перемещение, изменение размера, закрытие).

    • Управление окнами

  • Win + ↑ — развернуть окно на весь экран;
  • Win + ↓ — свернуть окно или восстановить его размер;
  • Win + ← / → — прикрепить окно к левой/правой половине экрана;
  • Win + Shift + ← / → — переместить окно на другой монитор;
  • Win + Home — свернуть/развернуть все окна, кроме активного.

    • Виртуальные рабочие столы

  • Win + Ctrl + D — создать новый виртуальный рабочий стол;
  • Win + Ctrl + ← / → — переключение между виртуальными рабочими столами;
  • Win + Ctrl + F4 — закрыть текущий виртуальный рабочий стол.

    • Работа с файлами и текстом (универсальные)

  • Ctrl + C — копировать;
  • Ctrl + X — вырезать;
  • Ctrl + V — вставить;
  • Ctrl + Z — отменить последнее действие;
  • Ctrl + Y — повторить отменённое действие;
  • Ctrl + A — выделить всё;
  • Ctrl + S — сохранить;
  • Ctrl + Shift + S — «Сохранить как…»;
  • Ctrl + O — открыть файл;
  • Ctrl + N — создать новый документ/окно;
  • Ctrl + W — закрыть текущее окно/документ;
  • Ctrl + F — поиск;
  • Ctrl + H — найти и заменить;
  • Ctrl + P — печать;
  • F2 — переименовать выделенный файл/папку;
  • Delete — удалить в корзину;
  • Shift + Delete — удалить без помещения в корзину;
  • F5 — обновить содержимое окна/страницы;
  • Alt + Enter — открыть свойства файла;
  • Alt + P — включить панель предпросмотра в Проводнике.

    • Форматирование текста (в редакторах)

  • Ctrl + B — жирный шрифт;
  • Ctrl + I — курсив;
  • Ctrl + U — подчёркивание;
  • Ctrl + L — выравнивание по левому краю;
  • Ctrl + E — выравнивание по центру;
  • Ctrl + R — выравнивание по правому краю.

    • Горячие клавиши для браузера (Chrome, Edge, Firefox, Яндекс Браузер)

  • Ctrl + T — открыть новую вкладку;
  • Ctrl + W — закрыть текущую вкладку;
  • Ctrl + Shift + T — восстановить закрытую вкладку;
  • Ctrl + Tab / Ctrl + Shift + Tab — перейти к следующей/предыдущей вкладке;
  • Ctrl + 1–8 — перейти к вкладке по номеру (1–8);
  • Ctrl + 9 — перейти к последней вкладке;
  • Ctrl + N — новое окно браузера;
  • Ctrl + Shift + N — режим инкогнито;
  • Ctrl + L или Alt + D — перейти в адресную строку;
  • Ctrl + Enter — добавить www. и .com к введённому адресу;
  • F5 или Ctrl + R — обновить страницу;
  • Ctrl + F5 — полное обновление (с очисткой кэша);
  • Ctrl + D — добавить страницу в закладки;
  • Ctrl + H — открыть историю посещений;
  • Ctrl + J — открыть список загрузок;
  • Ctrl + + / - / 0 — увеличить/уменьшить/сбросить масштаб;
  • Home / End — в начало/конец страницы;
  • Space / Shift + Space — прокрутка вниз/вверх на экран;
  • Alt + ← / → — назад/вперёд по истории посещений.

    • Скриншоты

  • Print Screen (PrtSc) — скопировать скриншот всего экрана в буфер обмена;
  • Alt + Print Screen — скопировать скриншот активного окна;
  • Win + Print Screen — сделать скриншот всего экрана и сохранить в папку «Изображения\Снимки экрана»;
  • Win + Shift + S — снимок части экрана («Ножницы»).

    • Функциональные клавиши (F1–F12)

  • F1 — вызвать справку (в большинстве программ);
  • F2 — переименовать (в Проводнике);
  • F3 — поиск (в Проводнике и других приложениях);
  • F4 — переход к адресной строке (в Проводнике), также открытие списка в поле ввода;
  • F5 — обновить;
  • F6 — переход между элементами окна;
  • F10 — активация меню программы;
  • F11 — полноэкранный режим;
  • F12 — «Сохранить как» (в некоторых редакторах), инструменты разработчика (в браузере).
